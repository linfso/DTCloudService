杜塔云服务项目
==============

A Cloud Service Products 
v0.0.1 
-----------------------------
create 2014-12-20 by linfso

本项目主要为基于node.js 云计算提供服务<br>
分为公司私有云和公共云服务

1.提供中间路由管理<br>
2.支持服务提供者动态注册服务，但必须遵循命名规范<br>
   接口名称<br>
   接口url地址<br>
   请求参数<br>
   响应结果<br>
   作者<br>
   描述<br>
   版本号<br>
3.服务使用者需要根据版本号和接口选择需要支持的服务<br>
