A Cloud Service Products 
v0.0.1 
-----------------------------
create 2014-12-20 by linfso

本项目主要为基于node.js 云计算提供服务
1.提供中间路由管理
2.支持服务提供者动态注册服务，但必须遵循命名规范
   接口名称
   接口url地址
   请求参数
   响应结果
   作者
   描述
   版本号
3.服务使用者需要根据版本号和接口选择需要支持的服务
